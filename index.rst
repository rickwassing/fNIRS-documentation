=======================
fNIRS Analysis Pipeline
=======================

This documentation website gives instructions on how to import, process, and analyse functional NIRS data using NIRStorm.

Version 1.0.0
-------------

Authors
-------
**Rick Wassing**, rick.wassing@sydney.edu.au, Woolcock Institute of Medical Research, The University of Sydney, Australia

Dependencies
------------
- Matlab, including the Curve Fitting Toolbox and the Signal Processing Toolbox
- `Brainstorm <https://neuroimage.usc.edu/brainstorm/Introduction>`_
- `NIRStorm <https://github.com/Nirstorm/nirstorm/wiki>`_
